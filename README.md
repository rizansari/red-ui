# RedUI - Redis DeskTop
<hr/>

> Cross platform Redis Desktop Manager built with Electron, Angular, Clarity UI and ioredis.

## Main Features
- Cross platform
- Supports List view and Tree view
- "Monitor" mode
- Display Redis "Info"
- JSON formatting for string
